# 叙事知识图谱

叙事不是孤立存在的，它们之间有支撑、挑战、衍生等关系。知识图谱帮你看到全景。

---

## 叙事层级结构

```
千年叙事（100年+）
├── 分工的消失
│   ├── 支撑叙事：程序员skill化
│   ├── 支撑叙事：专业壁垒崩塌
│   └── 挑战叙事：新分工形式出现？
│
五年叙事（3-7年）
├── 推理时代
│   ├── 支撑叙事：推理成本暴跌99%
│   ├── 支撑叙事：边缘计算崛起
│   └── 衍生叙事：AI应用成本趋零
│
当下叙事（1年）
├── 大模型军备竞赛
├── AI监管立法潮
└── 算法歧视争议
```

## 叙事关系类型

### 1. 支撑关系
叙事A为叙事B提供证据

**例子**：
- "推理成本暴跌" → 支撑 → "推理时代"
- "程序员skill化" → 支撑 → "分工消失"

**写作提示**：发现新事件时，想想它支撑哪个上层叙事

### 2. 挑战关系
叙事A质疑叙事B

**例子**：
- "新职业诞生速度 > 旧职业消失速度" → 挑战 → "分工消失"
- "蓝领工作难以自动化" → 挑战 → "AI替代论"

**写作提示**：叙事不是真理，要敢于提出挑战性证据

### 3. 衍生关系
叙事A延伸出叙事B

**例子**：
- "推理时代" → 衍生 → "AI应用成本趋零"
- "分工消失" → 衍生 → "全栈个体崛起"

**写作提示**：从大叙事推导出具体影响

### 4. 冲突关系
叙事A和叙事B看似矛盾

**例子**：
- "AI提高效率" vs "AI制造焦虑" → 冲突
- 解决：效率提升 ≠ 幸福感提升

**写作提示**：冲突不是问题，是深入分析的机会

## 叙事交叉分析

### AI叙事 × 经济叙事

**交叉点**：AI推理成本下降 × 全球化区块化

**衍生洞察**：
- AI让小国/小公司也能用顶尖能力
- 可能削弱大国的技术垄断优势
- 全球化从"资源全球化"转向"能力全球化"

### AI叙事 × 社会叙事

**交叉点**：AI替代工作 × 年轻人入门路径消失

**衍生洞察**：
- AI最先侵蚀的是"入门级工作"
- 22-25岁进入高AI暴露职业比率降14%
- 不是"AI抢工作"，是"AI抢练级区"

### 经济叙事 × 社会叙事

**交叉点**：主权债务陷阱 × 年轻人躺平

**衍生洞察**：
- 转型成本从隐形变显性
- 年轻人用"躺平"回应"承担转型成本"的期待
- 代际冲突的本质是"谁为转型买单"

## 使用方法

1. **发现新事件时**：
   - 在知识图谱中找到它的位置
   - 判断关系类型（支撑/挑战/衍生）
   - 思考是否有新的交叉点

2. **构建叙事时**：
   - 从当下叙事出发，链接到五年叙事、千年叙事
   - 引入相关学术概念
   - 寻找与其他叙事的交叉点

3. **深度分析时**：
   - 画出叙事之间的关系图
   - 找出冲突点，深入分析
   - 预测叙事的演化方向
