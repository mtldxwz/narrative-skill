# 叙事（Narrative）

> 从纷繁信息中提炼有规律可循的叙事，帮你看到表象背后的本质。

## 这是什么？

一个帮助叙事者从信息洪流中提炼有解释力的叙事的技能框架。

**信息爆炸时代，稀缺的不是信息，是"框架"。**

大多数人每天看到各种新闻、动态、热点——但他们只看到了"发生了什么"，没有看到"这背后意味着什么"。

叙事者不是讲故事，是**定义问题**。问题的价值往往大于答案。

## 核心叙事框架

### 千年叙事：分工的消失
人类社会因智能不足而分工。AI时代，当智力像自来水一样廉价，分工可能终结。

### 5年叙事：推理时代
训练时代已过，现在是推理时代。AI从"学习知识"转向"应用知识"。

## 输出格式

每天生成10条叙事，分为三个维度：

- **AI叙事（5条）** - AI技术、应用、监管、伦理
- **经济叙事（2条）** - 宏观经济、产业变迁、全球化
- **社会叙事（3条）** - 人口结构、文化心理、社会问题

每条叙事包含：
- 【叙事】一句话概括
- 【来源】多个信息来源
- 【概念】学术理论支撑
- 【我怎么看】个人观点

## 使用方法

### 方法一：人类使用
直接阅读 `SKILL.md` 和 `references/` 目录下的文件，参考叙事框架来分析信息。

### 方法二：AI使用
将 `SKILL.md` 作为prompt或skill加载给AI，让AI按照框架生成叙事报告。

### 方法三：定制使用
基于这个框架修改，加入你自己的叙事维度、学术概念、输出格式。

## 叙事者的本质

**叙事者不是讲故事，是定义问题。**

大多数人回答别人提出的问题：这个产品好不好？这个技术厉害不厉害？

叙事者提出问题本身。

**叙事者在塑造认知框架。**

当叙事被接受，人们不只是"知道了你的观点"，而是开始用你的框架看世界。

**每个时代都有它的核心叙事。**

- 工业时代：机器取代人力
- 信息时代：数据是新的石油
- AI时代：？——这个空白，就是机会
